What does this website ?

This website allows to anyone to get access to a gigantic movie database. This database lists the caracteristics of all movies ever created.

To get those informations, you just have to write in the search bar above, the title of the movie you're looking for. For more accuracy, please don't forget to use the year of creation/release.

What is its purpose ?

The idea of creating this website come from a training project with the purpose to improve web programming skills, such as the code languages HTML, JavaScript or CSS.

Now, this website just helps strangers requesting for some informations about their favourite movie.
